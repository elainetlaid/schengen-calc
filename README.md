## 🚀 快速上手教學

本工具用以計算 180 天內不超過 90 天的限制，並提供未來行程的試算功能。

### 🔸開始使用

* **新增紀錄**：點擊首頁的 **「📋 出入境紀錄管理」** > **「+ 新增」**，輸入您的歷史入境與出境日期。
* **查看狀態**：首頁日曆會自動標示停留期間，並在每一格日期右下角顯示該天「剩餘可停留天數」。
* **試算未來行程**：在首頁最下方，點擊 **「+ / -」** 或手動輸入您計畫待的天數，系統會自動算出最後一筆紀錄之後，**最快哪一天可以再次出發**。

---

### 🔸備份與還原資料 (重要！)

由於本工具為了保護個人隱私，所有資料皆儲存在您手機/電腦的**瀏覽器本地快取 (Local Storage)**，不會上傳至任何雲端伺服器。**若您更換手機、清除瀏覽器快取，資料將會消失。** 建議定期執行備份。

#### 💾 如何備份資料？

1. 點擊 **「📋 出入境紀錄管理」**。
2. 點擊底部的 **「💾 備份」** 按鈕。
3. 系統會自動將您的所有紀錄轉換成一段「加密代碼」並**自動複製到剪貼簿**。
4. 請將這段代碼貼到您的備忘錄、Line 記事本或 Email 給自己保存。

#### 🔄 如何還原資料？ (例如換新手機時)

1. 在新手機打開網頁，點擊 **「📋 出入境紀錄管理」**。
2. 點擊底部的 **「🔄 還原」** 按鈕。
3. 在彈出的對話框中，**貼上**您先前備份的那段加密代碼。
4. 點擊確定，您的所有歷史紀錄就會立刻出現在新手機上！

### 📱 推薦設定：加入主畫面

為了獲得像原生 App 一樣的體驗（全螢幕、無網址列），建議將此網頁「加入主畫面」：

* **iPhone (Safari)**：點擊瀏覽器下方的「分享」圖示（向上箭頭方框），選擇 **「加入主畫面」**。
* **Android (Chrome)**：點擊右上角三個點，選擇 **「安裝應用程式」** 或 **「加入主畫面」**。
